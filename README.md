# GLM Coding 自动抢购助手

Chrome浏览器扩展，每日10:00 CST自动抢购GLM Coding套餐（Lite/Pro/Max）。

GitHub仓库：https://github.com/PaidaxingW/glm-coding-snap-extension

## 快速开始

### 方法一：从GitHub安装（推荐）

```bash
# 克隆仓库到本地
git clone https://github.com/PaidaxingW/glm-coding-snap-extension.git
```

然后在Chrome中：
1. 打开 `chrome://extensions/`
2. 开启右上角**开发者模式**
3. 点击**"加载已解压的扩展程序"**
4. 选择刚才克隆的 `glm-coding-snap-extension` 文件夹
5. 安装完成！

### 方法二：直接下载

1. 访问 https://github.com/PaidaxingW/glm-coding-snap-extension
2. 点击绿色 **Code** 按钮 → **Download ZIP**
3. 解压ZIP文件
4. 按方法一的步骤3-5加载扩展

## 使用步骤

### 第一步：准备账号
1. 访问 https://bigmodel.cn/glm-coding
2. 登录你的智谱账号
3. **重要**：建议提前5-10分钟登录，确保抢购时已是登录状态

### 第二步：检查扩展状态
1. 点击浏览器工具栏中的扩展图标
2. 确认状态显示**"已开启"**
3. 查看**下次触发时间**是否为今天10:00 CST
4. 可点击**"立即触发测试"**按钮测试功能

### 第三步：等待自动抢购
- 扩展会在每天10:00 CST自动触发
- 抢购结果会通过桌面通知告知
- 扩展只点击购买按钮，**不会自动支付**，需手动完成

### 第四步：选择套餐（重要）
1. 打开扩展弹出面板，在**"目标套餐"**处选择 Lite / Pro / Max（三选一）
2. 也可以右键扩展图标 → **"选项"** 选择套餐类型
3. **只会抢购你选中的那一个套餐**，不会同时抢三个

### 第五步：配置（可选）
右键扩展图标 → **"选项"**，可设置：
- ✅ 选择抢购套餐（Lite/Pro/Max）
- ✅ 启用/禁用自动触发
- ✅ 提前触发时间（建议设置5-10分钟提前量）
- ✅ 启用/禁用桌面通知

## 功能特性

- ✅ **三选一**：Lite / Pro / Max 单选，只抢一个套餐
- ✅ 每日10:00 CST自动触发抢购（可设置提前量）
- ✅ 自动检测登录状态，未登录时提示
- ✅ 智能查找目标套餐购买按钮（多策略容错）
- ✅ 模拟人类操作延迟，降低被检测风险
- ✅ 桌面通知提醒抢购结果
- ✅ 弹出面板查看状态、手动触发测试
- ✅ 设置页配置套餐类型、自动触发、通知偏好

## 工作原理

1. **定时调度**：Chrome Alarms API在每天10:00 CST（UTC+8）触发
2. **页面交互**：Content Script在目标页面执行DOM操作
3. **登录检测**：检查页面登录状态
4. **按钮点击**：智能查找Pro套餐购买按钮并点击
5. **结果通知**：桌面通知告知抢购结果

## 常见问题

**Q: 为什么没有成功抢到？**
A: 成功率受库存、网络、并发等影响。本扩展只自动化点击流程，不保证100%成功。

**Q: 扩展没有反应？**
A: 检查：扩展是否启用、是否已登录bigmodel.cn、浏览器控制台是否有错误。

**Q: 可以修改触发时间吗？**
A: 在设置页面设置"提前触发时间"，例如设为5，则9:55开始尝试。

**Q: 会自动完成支付吗？**
A: 不会。只点击购买按钮，支付需手动完成，避免误操作。
